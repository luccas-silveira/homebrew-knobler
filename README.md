# homebrew-knobler

Tap do [Knobler](https://github.com/luccas-silveira/knobler) — Dynamic Island pro notch do Mac.

```bash
brew tap luccas-silveira/knobler && brew trust luccas-silveira/knobler && brew install knobler
```

App assinado ad-hoc (sem Developer ID). O `brew trust` é exigido pelo Homebrew 6+
em taps de terceiros; o cask remove a quarentena no install. Update: `brew upgrade knobler`.
